Software Packages

Explore the open-source tools and frameworks I've developed for computational archaeology, deep learning applications, and quantitative analysis.

PyPotteryInk

Deep learning for archaeological inking

A deep learning Python package for automating the digital inking process of archaeological pottery drawings. It transforms pencil drawings into publication-ready illustrations while preserving the original characteristics.

GitHub Repository

PyPotteryLens

Desktop app for pottery drawings recording

A Gradio desktop application for archaeological pottery drawings recording. It provides a comprehensive suite of tools for processing, detecting, and analyzing pottery fragments from scanned documents.

GitHub Repository

CQArchaeo

Kendall CQA results analysis

A Python package for calculating and plotting Kendall CQA results. It allows researchers to test numerical datasets using Kendall's formula (1974) efficiently.

GitHub Repository
